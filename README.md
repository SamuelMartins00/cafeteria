# ☕ Coffee Code

O **Coffee Code** é um projeto de site desenvolvido em HTML e CSS para uma cafeteria moderna.
A proposta do projeto é apresentar a marca, seus produtos e permitir que os clientes entrem em contato de forma simples e intuitiva.

---

## 🌐 Visão Geral

O site foi estruturado com foco em organização, semântica e experiência do usuário, contendo páginas informativas e um formulário funcional de contato.

---

## 📄 Páginas do projeto

* 🏠 **Home (index.html)**
  Apresentação da cafeteria, ambiente e destaques do cardápio.

* 👥 **Sobre (sobre.html)**
  Informações sobre a empresa, missão e valores.

* 📞 **Contato (contato.html)**
  Formulário para envio de mensagens e comunicação com o cliente.

---

## 🚀 Funcionalidades

* Estrutura semântica com HTML5
* Layout com Flexbox
* Navegação entre páginas
* Formulário de contato com validação básica
* Organização de arquivos em pastas

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* BOOTSTRAP
